xv6 项目安装

在课程仓库的 INSTALL-xv6.md 中，仅仅编写了 MacOS 下的实验初始化方法，接下来是简单的 Linux 系统初始化方法

Linux

项目仓库克隆

使用 git clone https://github.com/mit-pdos/xv6-rsicv 进行项目克隆（我们使用 rsicv 架构版本，而非课程仓库中MacOS环境下的x86架构版本，接下来的所有实验操作都在此版本上进行）

基础实验工具

使用包管理器进行安装

  • 虚拟机

    • qemu

  • 交叉编译工具

    • riscv64-linux-gnu-gcc

项目编译

构建前

构建后，新增了一个 fs.img 文件系统镜像文件

在项目根目录下运行 make qemu 即可，
在等待一段时间的构建与编译后，终端会显示 $ 提示符预示着已经进入 xv6* 系统的 qemu 界面，可以使用 ls 显示当前所有内建命令，使用 ctrl+a x (先 ctrl+a 再 x) 退出 qemu
