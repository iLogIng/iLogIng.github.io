SFML cloth simulation

使用 粒子弹簧模型 进行 运动学模拟

理论公式

位置 Verlet 积分

选用 Verlet 位置更新公式

Verlet 积分: x(t + dt) = 2x(t) - x(t - dt) + a(t)dt^2 [O(dt^4)]

公式推导

泰勒展开法

更新质点状态

  1. 假设粒子运动状态

    • 粒子的位置为 x(t)
    • 速度为 v = x’(t)
    • 加速度为 a(t) = F(x(t)) / m = x’’(t)
    • 以上三项公式都仅依赖于 位置

  2. 取时间步长为 dt ，分别在 t 时刻前后作三阶泰勒展开

    • x(t + dt) = x(t) + x’(t)dt + (1/2)x’’(t)dt^2 + (1/6)x’’’(t)dt^3 [+ O(dt^4)]
    • x(t - dt) = x(t) - x’(t)dt + (1/2)x’’(t)dt^2 - (1/6)x’’’(t)dt^3 [+ O(dt^4)]

  3. 将上述两式相加

    • x(t + dt) + x(t - dt) = 2x(t) + x’’(t)dt^2 [+ O(dt^4)]
    • 将上式进行整理 x(t + dt) = 2x(t) - x(t - dt) + x’’(t)dt^2 [+ O(dt^4)] 得到上述公式
    • 可见，该式表示，当前的位置仅依赖于上一时刻的位置，以及当前的加速度
    • 若需要速度，可是使用 中心差分估计
      • v(t) = ( x(t + dt) - x(t - dt) ) / 2dt [+ O(dt^2)]

点到线段距离

连线破坏事件

  • 使用 鼠标左键点击事件 确定将要被破坏的连线
  • 需要计算事件发生时光标位置与连线间的最短距离

公式推导

  • 假设计算 点P 到 线段AB 的距离
投影公式

  1. 投影长度计算

    • proj_len = |AP| * cos<AP, AB>
      • 余弦公式 cos<AP, AB> = AP*AB / |AP|*|AB|
    • 将余弦公式带入后可得 proj_len = AP * AB / |AB|

  2. 投影参数 t (投影相对于AB的长度)

    • t = proj_len / |AB|
    • 带入后可得 t = AB * AP / |AB| * |AB|

  3. 计算A到投影点的向量

    选择A是由于在上述公式推导中，是以 AB AP 向量为基础进行推导的
    同时，在计算后可以知道，参数t是 APproj向量 相对于 AB向量 的方向与大小

    1. t < 0.f
      • Pproj 更靠近A侧
      • return |AP|
    2. t > 1.f
      • Pproj 更靠近B侧
      • return |BP|
    3. 0.f <= t <= 1.f
      • Pproj 在线段AB上
      • return |PprojP| 这时 PprojP 就是P到AB的垂段
