Raylib 杂记

Raylib 简介
Raylib 是一个 简单、易用且跨平台 的游戏开发库，无外部依赖，
特别适合用来学习图形学、游戏编程和快速制作原型。
它完全使用 C 语言写成（同时也提供了超过 50 种语言的绑定），
核心设计理念是“没有复杂依赖，开箱即用”。

2D坐标，原点在屏幕左上角

  • x: 指向屏幕横向向右
  • y: 指向屏幕纵向向下

3D坐标，采用右手系

  • x: 指向屏幕横向向右
  • y: 指向屏幕纵向向上
  • z: 指向屏幕外侧

OpenGL 相关模块

GLFW

Graphics Library Framework
是一个专门为 OpenGL 设计的 跨平台窗口和输入处理库

  • 功能：创建窗口、创建 OpenGL 上下文、处理键盘鼠标手柄输入、管理时间等，彻底屏蔽操作系统层面的差异
  • 目前版本的 Raylib 不再依赖 GLFW，目前Raylib已拥有窗口和输入管理功能

GLSL

OpenGL Shading Language
是运行在 GPU 上的， 着色器编程语言，语法类似于 C

  • 作用：现代图形渲染管线是“可编程”的，GLSL 就是用来编写这些运行在显卡上的程序。最常见的是顶点着色器（处理每个顶点的位置变换）和片元着色器（计算每个像素的颜色）。
  • 使用前提：将 顶点着色器 放在 .vs 命名的目录下，将 面着色器 放在 .fs 命名的目录下

在 Raylib 中可以使用 LoadShader() 加载 <name>.glsl 文件，并使用 BeginShaderMode() 应用该着色器程序

<rlgl.h>

Raylib OpenGL

对 OpenGL 渲染管线 进行了轻量封装，提供了一个操作 OpenGL层 的方法，是 Raylib 与 OpenGL 的一个中间层

API与数据类型设计

Function 函数

每个 Raylib 函数命名都遵循以下语法结构：
<动词><主语><补语>
或者说
<动作行为><对象><属性/状态>
即：
在 <某物> 的 <某 属性/状态> 上，做 <动作行为>

Raylib 函数大多被设计为值传递函数，小部分需要传递引用

Enumerator 枚举类型

Raylib 定义了 21种枚举类型 (v5.1)
各个枚举类中的枚举值，一般都将该类的 大写全称 作为 前缀
随着库的更新，可能会添加相应的枚举状态

没有函数指向这些枚举类型，所有相关参数都被定义为 int 类型，在程序中仅作为定义值使用
