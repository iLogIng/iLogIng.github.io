配置国内模型

配置文件： ~/.claude/settings.json

配置说明

  • env.ANTHROPIC_BASE_URL: 国内大模型的API端点URL
  • env.ANTHROPIC_AUTH_TOKEN: 您的API密钥
  • model: 默认使用的模型名称
  • custom-model: 自定义模型配置
    • name: 模型名称（自定义）
    • endpoint: API端点URL
    • api_key: API密钥
    • model_type: 模型类型（通常为”code”）

提示词工程

事先声明

  1. AI 是 外置大脑 而非个人的替身

    • 要保证项目代码的最终决策权在我们自己，你需要不断引导它分析约束、评估风险、给出可选方案，而不是直接接受第一个答案。

  2. 好的架构是分视图、分阶段、分模块的

    • 你必须在提示词里显式要求 AI 从不同架构视图（逻辑、过程、开发、部署、数据）思考，避免它过早陷入细节。
    • 你最好能事先规范化项目的顶层设计，再与AI进行总体的设计讨论，逐步深入。

  3. 每一次讨论都是一次 架构评审

    • 把提示词设计成微小的评审会议：输入上下文、假设，要求 AI 输出分析、质疑、建议。
    • 但是，绝不能被AI牵着鼻子走，在整理AI的建议时需要自己的思考，纠正AI偏离预期设计的想法。
