网页框架初始化

是网页框架的主文件夹名

初始化的文件结构

  • _config.yml
    • 网站配置文件
  • package.json
    • 应用程序信息
  • scaffolds
    • 模板 文件夹，新建文章时，Hexo根据该模板创建文件
  • source
    • 资源文件夹。存放用户资源。除 _posts 文件夹外，其它以 下划线_ 开始的文件都将被忽略。
    • Markdown 和 HTML 文件会被解析并放到 public 文件夹，其他的文件将被拷贝至该目标。
  • theme
    • 主题文件夹。Hexo根据主题生成静态文件。
